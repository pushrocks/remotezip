## Usage
Use TypeScript for best in class instellisense.

```javascript
import * as smartarchive from 'smartarchive'
smartarchive.get({
    from:"https://example.com/example.zip",
    toPath:"/some/local/absolute/path",
}).then(/*...*/)
```
